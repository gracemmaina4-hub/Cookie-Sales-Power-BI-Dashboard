# Cookie-Sales-Power-BI-Dashboard
This project is an interactive Power BI dashboard designed to analyze cookie sales performance, customer behavior, and revenue trends.
The dashboard provides a comprehensive view of sales data across different cookie types, locations, and time periods. It enables data-driven decision-making by highlighting key performance indicators and trends.
Insights Delivered
1.Identifies top-performing cookie products
2.Tracks revenue growth over time
3.Analyzes customer purchasing behavior
4.Helps optimize product and sales strategy
